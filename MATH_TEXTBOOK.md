# 数学I・A・II・B ミニ教科書

この教科書は、アプリで問題を解く前に「単元の考え方」を短くつかむための読み物です。
長い説明を全部読むより、まずは型を理解してから演習へ入ることを狙っています。

## 使い方

1. 今日は 1 単元だけ読む
2. 例題の考え方を確認する
3. すぐにアプリか練習問題集で 3 問解く

## 数学I

### 二次関数

二次関数は、まずグラフの形を頭に置く単元です。
`a(x-h)^2+k` の形なら、頂点は `(h, k)`、軸は `x=h` です。

最大最小の問題では、毎回この順で見ます。

1. 軸はどこか
2. 定義域はあるか
3. 軸が範囲内か
4. 頂点と端点のどれを比べるか

例:
`f(x)=(x-2)^2-3, 0<=x<=5`

このとき軸は `x=2` で範囲内です。
上に開くので最小値は頂点の `-3`、最大値は端点比較で決めます。

### 三角比

三角比は、`値` と `符号` を分けて考えると整理しやすいです。

- まず基準角の値を見る
- 次に象限で符号を見る

たとえば `cos120°` は、
基準角 `60°` の `cos` の値 `1/2` を出して、
第2象限なので負にして `-1/2` です。

### 図形と計量

図形と計量では、図をただ眺めるより「どの公式の場面か」を先に決めます。

- 2辺とその間の角 -> 面積公式
- 3辺が見えている -> 余弦定理
- 角と向かい合う辺の対応 -> 正弦定理

まず図の条件を日本語で言い換えることが大切です。

## 数学A

### 場合の数

場合の数は「順番が関係あるか」が最重要です。

- 順番あり -> 順列
- 順番なし -> 組合せ

ここを外すと計算が合っていても全体が崩れます。

例:
5人から2人を選ぶ
これは並べないので `5C2=10`

5人から2人を選んで並べる
これは順番ありなので `5P2=20`

### 確率

確率は `有利な場合 / 全体の場合` です。
ただし、いきなり式に飛ばず、まず全体を数えることが大切です。

余事象が使えるときは、直接数えるより楽になります。

例:
「少なくとも1回当たる」
これは `1回も当たらない` を引く方が速いことが多いです。

## 数学II

### 指数・対数

対数で迷ったときは、指数の形へ戻すのが基本です。

`log_a b = x`
なら
`a^x = b`

この変換ができると、値の問題も方程式も整理しやすくなります。

また、対数では定義域も重要です。
真数は正、底は正かつ1でないことを忘れないようにします。

### 微分法

微分法では、求めたいものを先に決めるのが大切です。

- 接線の傾き -> 微分して代入
- 増減 -> 導関数の符号
- 極値 -> 増減の切り替わり

例:
`f(x)=x^2-4x+1` の `x=3` での接線の傾き

`f'(x)=2x-4`
ここへ `x=3` を代入して `2`

順番は必ず
`微分 -> 代入`
です。

## 数学B

### 数列

数列では、まず「等差か等比か」を見ます。

- 一定の差 -> 等差
- 一定の比 -> 等比

等差数列の一般項:
`a_n = a_1 + (n-1)d`

等比数列の一般項:
`a_n = a_1 r^(n-1)`

`n-1` を落とすミスが多いので、ここは毎回確認します。

### ベクトル

ベクトルは、図形を成分で扱えるようにする単元です。

内積は
`a・b = x1x2 + y1y2`

で計算できます。
図形問題でも、条件を成分の式へ翻訳できるかが勝負です。
